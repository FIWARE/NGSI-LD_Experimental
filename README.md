# NGSI-LD_Wrapper

[![Build badge](https://img.shields.io/travis/jmcanterafonseca/NGSI-LD_Wrapper.svg "Travis build status")](https://travis-ci.org/jmcanterafonseca/NGSI-LD_Wrapper/)

NGSI-LD Wrapper implemented in Scala

## How to build / test

### Prerequisites

* Java 8
* Scala runtime
* SBT build tool

```console
sbt compile
sbt jetty:start
```

```console
sbt test
```
